# Notenmanagement – GIBZ

Projekt: Notenmanagement-Software für die Prorektoren der GIBZ
Team: Nikola Matic, Melih Yasar, Ben Birrer
Klasse: INFA1a

Dieses Projekt ist eine innovative dreischichtige Blazor-Anwendung, die entwickelt wurde, um den Prozess der Notenverwaltung an der GIBZ zu optimieren. Unsere Software integriert Backend, Datenbank und Frontend nahtlos, um Lehrpersonen eine einfache, einheitliche Möglichkeit zu bieten, Noten direkt an den Rektor zu übermitteln.

Hauptfunktionen:

▪️Einheitliche Erfassung und Verwaltung von Schülernoten

▪️Sichere Speicherung und Abruf von Daten über das Backend

▪️Intuitive Benutzeroberfläche für Lehrpersonen

▪️Optimiert für einfache Integration in bestehende Schulprozesse

Technologien:

▪️Blazor (Frontend & Backend)

▪️SQL-Datenbank

▪️.NET Api als Backend

▪️HTML / CSS für benutzerfreundliches Design

▪️Zwei Faktor Authentifizierung mit Google SMTP Server

Dieses Projekt wurde speziell für den Einsatz an der Schule entwickelt und unterstützt Lehrpersonen und die Schulleitung dabei, den Notenverwaltungsprozess effizienter und übersichtlicher zu gestalten.
